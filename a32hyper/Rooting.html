Hướng Dẫn Root Android

Root là quá trình lấy quyền quản trị tối cao (Superuser) của thiết bị Android. Quá trình này thường được sử dụng để tùy biến sâu hệ điều hành hoặc cài đặt các ứng dụng yêu cầu đặc quyền hệ thống.

⚠️ LƯU Ý QUAN TRỌNG:

Thiết bị của bạn bắt buộc phải được Unlock Bootloader trước khi thực hiện. Quá trình này có thể làm mất bảo hành và xóa sạch dữ liệu của thiết bị.

1. Root bằng Magisk (Phổ biến)

  1. Tải về file Magisk.zip và Magisk Manager.apk từ trang chính thức của Magisk.
  2. Khởi động thiết bị vào chế độ Custom Recovery (TWRP/OrangeFox...).
  3. Flash file Magisk.zip thông qua bộ nhớ hoặc USB OTG.
  4. Khởi động lại điện thoại và cài đặt Magisk Manager.apk để cấp/quản lý quyền root.

2. Root bằng KernelSU (Hiện đại)

Lưu ý về KernelSU:

Yêu cầu thiết bị phải có Kernel hỗ trợ (GKI) và có thể không tương thích với các dòng máy cũ.

  1. Tải ứng dụng KernelSU (file APK) từ trang chủ để kiểm tra trạng thái hỗ trợ của thiết bị.
  2. Tìm và tải Custom Kernel được tích hợp sẵn KernelSU dành riêng cho mã máy và phiên bản ROM chính xác của bạn (thường tìm trên XDA Developers).
  3. Flash Custom Kernel này qua Custom Recovery.
  4. Khởi động lại và mở app KernelSU để quản lý quyền.

3. Root bằng APatch (Giải pháp mới)

Lưu ý về APatch:

Chỉ hỗ trợ chip kiến trúc ARM64 và phiên bản Linux Kernel từ 3.18 đến 6.12.

  1. Tải và cài đặt ứng dụng APatch Manager từ GitHub chính thức.
  2. Copy file boot.img gốc (trích xuất từ ROM stock đang dùng) vào bộ nhớ máy.
  3. Mở APatch, thiết lập SuperKey (Mật khẩu bảo mật từ 8-63 ký tự, tránh app độc hại tự chiếm quyền).
  4. Chọn file boot.img gốc và nhấn Patch để tạo ra file boot mới đã được vá.
  5. Flash file boot đã patch vào máy thông qua chế độ Fastboot trên máy tính (fastboot flash boot_name.img).
  6. Khởi động lại và kiểm tra quyền root trong ứng dụng APatch Manager.
